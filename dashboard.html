Executive Monitoring View

CSV를 올리면 위험한 흐름부터 바로 보이는 보안 대시보드

공격 유형 자동 분류, 위험도 우선 정렬, 시간대 차트, 상위 공격자 IP, 필터, CSV/PDF 내보내기를 한 화면에 묶었습니다. 경영진이 보기 쉽게 기술 용어는 줄이고 “무슨 일이 있었는지” 중심으로 보여줍니다.

한 번에 보는 흐름
업로드 → 분류 → 보고
CSV 업로드 후 자동으로 핵심 지표, 차트, 상세 근거, 대응 권고가 갱신됩니다.
업무 방식
기술보다 사용성
색상은 위험도 중심, 설명은 친절하게, 정리 방식은 경영진 보고 기준으로 맞췄습니다.

파일 업로드

드래그앤드롭과 파일 선택을 모두 지원합니다. 영문 컬럼명, 일부 누락 컬럼, 빈 행, 중복 행도 최대한 정리해서 읽습니다.

CSV 파일을 드롭하거나 선택하세요

`timestamp/time`, `source_ip/src_ip`, `path/url/url_path`, `status/status_code/response_code`가 있으면 가장 안정적으로 동작합니다.

대기 중 파일을 선택하거나 샘플 불러오기를 눌러 시작하세요.
파일을 읽지 못했습니다 CSV 형식을 확인해 주세요.
위험도별 색상 빨간색은 즉시 대응이 필요한 항목, 주황색은 빠른 조치가 필요한 항목입니다.
필터와 상세 보기 유형, 위험도, 시간대로 좁혀 보고, 사건 카드를 누르면 자세한 근거를 따로 보여줍니다.
보고서 출력 필터 결과를 CSV로 다운로드하고, 요약 PDF도 바로 생성할 수 있습니다.

보기 필터

보고 대상에 맞게 유형, 위험도, 시간대를 바로 좁혀볼 수 있습니다.

요약 지표

지금 업로드한 파일에서 가장 먼저 봐야 할 숫자만 추렸습니다.

위험한 이벤트

자동 분류된 사건을 위험도 순으로 보여줍니다. 카드를 누르면 오른쪽 상세 보기가 바뀝니다.

상세 보기

선택한 사건의 범위, 주요 근거, 대표 경로를 요약합니다.

위협 유형 분류

유형별 비중과 심각도 색상을 함께 보여줍니다.

공격자 IP 분석

많이 나타난 IP와 출처 분포를 함께 보여줍니다.

아래 분포는 IP 대역 기준의 간단한 출처 추정입니다.

시간대별 위험 흐름

시간대 막대를 클릭하면 해당 구간만 바로 필터링됩니다.

정상/노이즈 분리

헬스체크나 정적 자원 요청을 따로 떼어내야 사건이 더 선명하게 보입니다.

대표 근거

원본 로그를 열지 않고도 바로 설명할 수 있는 행만 뽑습니다.

즉시 대응 권고

현재 결과를 기준으로 먼저 실행할 조치만 남겼습니다.
